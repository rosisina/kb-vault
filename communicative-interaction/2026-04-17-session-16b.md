**원인 1 — TypeScript `detectLayer` 한계:**
- 기존: `layer 1 | first layer` 패턴만 인식
- 문제: `3.2.1. Manipulation...`, `3.3.1. Manipulations...` 등 섹션 번호 prefix 미인식

**원인 2 — Python `detect_layer` 오탐:**
- 기존: title에 `"first"` 단어만 있으면 Layer 1으로 인식
- 문제: `"2019. Environment... First, both environments are..."` → false positive Layer 1
- 결과: `s074`가 Layer 4 영역 안에 있음에도 Layer 1로 태깅

**수정 내용:**

| 파일 | 수정 내용 |

### 설계 확정: Search 맥락 답변 강화 (A+C 혼합)

**배경:** 현재 한계 = BM25 키워드 매칭 + atom 목록 그룹화, 질문 의도 미이해

**A+C 정의:**
- **A (Client BM25):** 기존 클라이언트 사이드 atom 스코어링 — $0, 즉시
- **C (사전 계산 JSON):** 주요 질문 50개 → Claude API 오프라인 생성 → `assets/preset-answers.json` — $2~5 1회성

**추가 기능 (A 강화):**
- 답변 하단 관련 균열(F-SC/F-CE 등) 자동 연결 섹션
- 7층 인과 체인 미니 시각화 (L2→L3→L4→L5 등)
- 인물 검색 → 인물 허브 뷰 연결

**선결 조건:**
- `detail.json` + `graph.json` 재빌드 필요 (`persons[]` 필드 현재 비어 있음)

**예상 소요:** 세션 α (0.5) 데이터 재빌드 + 세션 β (1.5) A+C 구현
