## 4. 논문 원문 핀치줌 — 4회 시도

### 시도 1: transform: scale()
**문제**: 줌인 가능, 줌아웃 불가 — 원래 DOM 경계 밖 터치 이벤트 차단

### 시도 2: font-size 스케일링
**문제**: 손 떼면 스케일 원상복구 (snap-back threshold 오류)

### 시도 3: CSS zoom 속성
**문제**: 삼성 갤럭시에서 확대·축소·이동 전체 미작동

### 시도 4: viewport meta 조작 (ngOnInit)
```typescript
vp.setAttribute('content', 'width=device-width, initial-scale=1, user-scalable=yes, minimum-scale=0.5, maximum-scale=5');
```
**문제**: `.pv-modal`이 `position: fixed` 오버레이로 뷰포트 전체를 덮어 브라우저 뷰포트 줌이 도달하지 못함

### 시도 5 (성공): NgZone 외부 JS 터치 이벤트 직접 처리

**파일**: `document-viewer.component.ts`

```typescript
private attachPinchZoom(): void {
  const el = document.querySelector<HTMLElement>('.pv-content');
  if (!el) return;

  this.zone.runOutsideAngular(() => {
    const onMove = (e: TouchEvent) => {
      if (e.touches.length !== 2) return;
      e.preventDefault();  // 브라우저 간섭 차단
      const d = this.pinchDist(e);
      if (this.lastPinchDist > 0) {
        this.pinchScale = Math.min(4, Math.max(0.5, this.pinchScale * (d / this.lastPinchDist)));
        el.style.fontSize = `${this.pinchScale * 100}%`;
      }
      this.lastPinchDist = d;
    };
    el.addEventListener('touchmove', onMove, { passive: false });
    // touchend: lastPinchDist만 0 리셋, pinchScale 유지 (snap-back 없음)
  });
}
```

**핵심 설계**:
- `NgZone.runOutsideAngular()` — Angular change detection 간섭 없음
- `{ passive: false }` + `e.preventDefault()` — 브라우저 기본 제스처 완전 차단
- `pinchScale` 필드 영구 유지 — 손 떼도 리셋 없음
- `font-size` 방식 — 텍스트 리플로우, 수평 오버플로 없음
- `overflow: scroll` — 확대 후 상하좌우 패닝 가능

**James 검증**: "확대 / 축소 가능" ✅

## 5. 논문 원문 모바일 레이아웃 수정

### 문제

모바일 SCSS에서 숨기려던 클래스명 오류:
- `.pv-sidebar` (존재 안 함) → 실제: `.pv-toc`
- `.pv-annot` (존재 안 함) → 실제: `.pv-atoms-panel`

결과: TOC 패널이 모바일에서 계속 노출되어 콘텐츠 가림. atoms 패널도 하단에 중복 노출.

### 수정

**파일**: `document-viewer.component.scss`

```scss
@media (max-width: 640px) {
  .pv-toc { display: none !important; }
  .pv-atoms-panel { display: none !important; }
  .pv-content {
    width: 100%; max-width: 100%;
    padding: 14px 14px 40px;
    overflow: scroll;
    flex: 1;
    touch-action: pan-x pan-y;
  }
}
```

**결과**: 모바일에서 문서 본문만 전체 너비 표시. TOC/atoms 패널 숨김.

## Key Decisions

1. **삼성 갤럭시 핀치줌** — fixed 오버레이 내부에서는 뷰포트 메타 방식 무효. `NgZone.runOutsideAngular()` + `{ passive: false }` + font-size 조합이 유일하게 작동
2. **패스워드 게이트** — `localStorage` 기반 인라인 JS, Angular 로드 이전에 실행 (DOMContentLoaded)
3. **CSS 클래스명 불일치** — 이전 세션에서 리팩터링으로 클래스명 변경됐으나 모바일 미디어쿼리가 미반영. 항상 실제 HTML 클래스명 대조 필요
