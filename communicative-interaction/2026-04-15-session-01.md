## Session Summary

Landing 첫화면을 Wikipedia essentialism 원칙 기반으로 전면 재설계. 2-column 레이아웃에서 "Proof Constellation" 정칠각형 중심 구조로 전환. 사용자 시선 집중, 심리적/감성적 친화력, one-stop 자기증명 시스템의 첫인상 최적화.

## Key Decisions (James 승인)

### 1. 7 Layers 관계 모델 수정
- **이전:** 각 층위가 다음 층위의 "전제조건" (선형)
- **수정:** 각 층위는 **상호 연관된 독립적 범죄** (N:N, 딥러닝 구조 비유)
- **근거:** Layer 5 허위 신고는 Layer 1 발각 방지 반응, Layer 7 기소유예는 전체 정당화 등

### 2. Wikipedia 구체 패턴 → Proof Constellation
- 중앙 로고 + 7 layers 정칠각형 배치 + SVG N:N 연결선
- 우측에 5개 도구 사슬(Chain Menu): Guided Proof, 증명 체인, 균열, 필터, 사건 개요
- 하단에 검색창 + 질문 예시 3개
- **3요소 구조:** 제목 + Heptagon(+Chain) + 검색

### 3. 균열 용어 수정
- "공식 서사" → **"국방 조직의 공식 서사"** (범죄 조직의 서사임을 명시)

### 4. 메트릭 제거
- 입증률/발견사항/균열/증거기록 4개를 화면에서 제거
- Guided Proof/균열 호버 카드에 통계 분산

### 5. 시각 강화 (방안 B+C)
- **B: 정칠각형 외곽선** — 로드 후 점선 페이드인, 호버 시 실선 전환
- **C: 중심 파동** — 로고에서 동심원 3개 순차 확산 후 소멸
