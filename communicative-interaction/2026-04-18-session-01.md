## 2. 사건 증명 문서 메뉴 (증거 탭)

### 구현
- Landing Case Overview → 증거(Evidence) 탭: Layer 필터 칩(L1–L7) + Atom 카드 목록
- Atom 카드 클릭 → proof-shell 전환 + atom 상세 자동 열림
- `getEvidenceAtoms(layer)`: layer별 최대 8개 atom 반환

### 핵심 코드
```typescript
// landing.component.ts
getEvidenceAtoms(layer: number): GraphNode[] {
  return this.graphData.getNodes(layer).slice(0, 8);
}
onEvidenceAtomClick(atomId: string, event: Event): void {
  event.stopPropagation();
  this.atomSelect.emit(atomId);
}
```

## 3. 이전 화면 복귀 화살표 수정

### 문제
- atom 카드 클릭 진입 → navTrail 소진 후 랜딩 복귀 수단 없음
- tool 클릭(Guided/Chain/Fractures/Filter) 진입 → 복귀 수단 없음

### 해결
- `onTrailBack()`: trail 1개 이하일 때 `onBackToLanding()` 호출
- `navTrail().length === 0`일 때 `.landing-back-bar` 상시 표시

## 5. 모바일 반응형 M-1 (Landing)

### 접근
- 기존 `transform: scale(0.7)` 해킹 제거
- `@media (max-width: 600px)`: `.main-area` 숨김, `.mobile-main` 표시

### 구현
- **Layer 그리드**: 7개 버튼을 2열로 배치 (색상+번호+이름)
- **도구 칩 행**: Guided/Chain/Fractures/Filter/Paper 수평 스크롤
- **사건 개요**: 사건/은폐/증거/구조/함의 탭 + 내용 펼침 (모바일 전용)

### 우측 잘림 수정 (후속)
- `.landing-page`에 `box-sizing: border-box; overflow-x: hidden` 추가
- `.mobile-main`, `.mobile-layer-grid`, `.mobile-tool-row`에 `box-sizing: border-box` 적용
- `max-width: 420px` 제거 → `width: 100%`로 통일

## 6. 모바일 반응형 M-2 (proof-shell)

### 문제
- `overflow-y: visible` → 스크롤 불가
- 상단 back bar 높이 미반영 → 레이아웃 깨짐
- 탭바 수평 오버플로

### 해결 (flex 기반)
```scss
.shell { display: flex; flex-direction: column; height: 100dvh; overflow: hidden; }
.proof-layout { flex: 1; height: 0; overflow: hidden; }
.panel-center { overflow-y: auto; overflow-x: hidden; height: 100%; padding: 16px 12px; }
```

### proof-body 탭바
```scss
.view-mode-toggle { overflow-x: auto; flex-wrap: nowrap; }
.mode-btn { flex: 0 0 auto; white-space: nowrap; }
```

## 7. 면책 고지 법적 검토 및 정비

James의 요청으로 About 모달 전문을 법적 위험 관점에서 검토 후 4항목 수정:

| 항목 | 변경 전 | 변경 후 | 이유 |
|---|---|---|---|
| 시스템 개요 | "조직적 은폐를 **입증**하기 위해" | "조직적 은폐 **의혹을 기록·제시**하기 위해" | 사법 판단 전 단정적 표현 위험 |
| 영문 번역 | "법적 효력 있는 원전은 한국어" | "법적 판단 근거 되지 않음" | 영문 독자 법적 조치 방어 |
| 공익 목적 | 없음 | "공익적 진실 규명 및 공적 기록 보존" 추가 | 명예훼손 위법성 조각 근거(형법 §310) |
| L6/L7 표현 | "사기 수사·범죄자 낙인" | "수사 적정성 문제·불이익 처분" | 직접적 명예훼손 표현 완화 |

### 추가 수정
- "296개의 원자 단위 클레임이 Luhmann 체텔카스텐 방식으로..." → "원자 단위 증명 주장(클레임)이 서로 연결되어 있으며..." (기술 용어 제거)
