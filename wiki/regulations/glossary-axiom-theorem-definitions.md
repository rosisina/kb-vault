# 용어 정의집 — 정의(Definition), 공리(Axiom), 정리(Theorem), 공준(Postulate) 체계

**Source:** raw/08. 용어 변천사/(220923 용어의 정의 및 설명 (Book Ver 2.9 7Layer).pdf (부록 15, 정의 01~36+)
**Layer:** 전 Layer 공통 — 용어의 공식 정의를 제공하는 참조 문서
**성격:** claim atom이 아닌 **reference hub** — 개별 정의는 주장이 아니라 용어 해설

## 개요

이 문서는 한지훈이 2022.9.23에 정리한 **책(raw/01)과 반박 문서(raw/05 20220929)에서 사용하는 모든 핵심 용어의 공식 정의집**이다. 스피노자의 『에티카(Ethica)』 방식을 따라 **정의(Definition)·해명(Explanation)·공리(Axiom)·공준(Postulate)·정리(Proposition)·증명(Demonstration)·따름정리(Corollary)·주석(Scholium)** 체계로 구성되어 있다.

## 논증 구조 정의 (정의 01~07)

| # | 용어 | 정의 |
|---|---|---|
| 01 | **정의(Definition)** | 존재자의 본질을 표현하는 것 / 용어의 뜻을 설정하는 것 |
| 02 | **공리(Axiom)** | 자명한 진리 또는 별도의 증명 없이 참된 것으로 인정되는 명제. 일반 원리를 표현하며 개별 사물에 대한 진리가 도출되지는 않는다 |
| 03 | **해명(Explanation)** | 어떠한 것을 좀 더 분명히 하기위한 진술이나 설명 |
| 04 | **공준(Postulate)** | 공리와 마찬가지로 암묵적으로 옳은 것으로 받아들이는 명제. 단, 특정한 학문 또는 특정한 분야에만 적용 |
| 05 | **정리(Proposition)** | 증명을 통해서 진리임이 입증된 명제. 6가지 죄명과 이에 대한 추측과 논박의 근간 |
| 06 | **따름정리(Corollary)** | 어떤 정리로부터 파생되어 나온 정리로서 별도의 증명을 필요로 한다 |
| 07 | **주석(Scholium)** | 정리, 증명, 따름정리 등을 통한 논증과 별도로 사태를 좀 더 분명히 설명하기 위한 보충 설명 |

## Key Takeaways

- [타당성] 정의 01~07은 **스피노자 『에티카』 방식의 논증 체계**를 정의 — 이 체계가 반박 문서(raw/05 20220929)와 책(raw/01)의 논증 구조 전체를 관통한다. / Definitions 01-07 establish the Spinoza-style argumentation framework used throughout all defense documents.
- [타당성] 정의 08~09(C/S, Active-X)는 **舊KIATIS의 기술 아키텍처**를 공식적으로 정의하며, Layer 1 핵심 취약점(VPN 없는 DB 직접접속)의 기술적 근거를 제공한다. / Definitions 08-09 formally define the technical architecture underlying the Layer 1 vulnerability.
- [타당성] 정의 17~34의 **형법 조문 정의**는 6개 혐의 각각의 법적 구성요건을 명시하여, 검찰의 혐의 적용이 구성요건을 충족하는지 검증하는 기준을 제공한다. / Criminal code definitions (17-34) provide the legal elements test for each of the 6 charges.
- [진리성] 정의 07 설명1은 한지훈이 **결재라인에서 고의적으로 제외**되었다는 사실을 용어 정의 문서에서까지 기록 — 용어 해설을 넘어 **claim 성격**의 사실 진술이 포함되어 있다. / Definition 07 Explication 1 contains a factual claim about deliberate exclusion from the approval chain.
